Angular Pipes Demo

Raw Date: {{ presentDate }}

Just displays the raw JavaScript Date object

Default Date Pipe: {{ presentDate | date }}

Formats the date using Angular's default locale format

Date Formats

Short date: {{ presentDate | date:'shortDate' }}

Short format (e.g., 1/14/26)

Full date: {{ presentDate | date:'fullDate' }}

Full format (e.g., Tuesday, January 14, 2026)

Custom format: {{ presentDate | date:'M/dd/yyyy' }}

Custom month/day/year format

Time (hours & minutes): {{ presentDate | date:'h:mm' }}

Shows only hour and minute

Text Transform

Uppercase: {{ presentDate | date:'fullDate' | uppercase }}

Converts formatted date to uppercase letters

Lowercase: {{ presentDate | date:'medium' | lowercase }}

Converts formatted date to lowercase letters

Async Pipe

Live time: {{ time$ | async | date:'mediumTime' }}

Subscribes to observable (time$) and formats it as time

Currency Pipe

Default: {{ price | currency:'':true }}

Formats number as currency without a symbol

INR: {{ price | currency:'INR' }}

Formats number as Indian Rupee currency

Slice Pipe (Fruits)

Start index (2): {{ Fruits | slice:2 }}

Returns array starting from index 2 to end

Start–end (1–4): {{ Fruits | slice:1:4 }}

Returns array from index 1 up to (not including) 4

Negative index (-2): {{ Fruits | slice:-2 }}

Returns last 2 elements of array

Negative range (-4 to -2): {{ Fruits | slice:-4:-2 }}

Returns array elements from -4 to -2 index

Number Pipe

Format 3.1: {{ decimalNum1 | number:'3.1' }}

Min 3 integer digits, 1 decimal

Format 2.1–4: {{ decimalNum1 | number:'2.1-4' }}

Min 2 integer digits, 1–4 decimals

Percent Pipe

Raw value: {{ completionRate }}

Shows raw numeric value

Percent (default): {{ completionRate | percent }}

Converts number to percentage (default format)

Percent (1 decimal): {{ completionRate | percent:'1.1-1' }}

Percentage with 1 decimal

Percent (2 decimals): {{ completionRate | percent:'1.2-2' }}

Percentage with 2 decimals

Title Case Pipe

Original: {{ titleText }}

Shows raw string

Title case: {{ titleText | titlecase }}

Capitalizes first letter of each word

Json Pipe

{{ user | json }}
Converts object into JSON string for display

Key Value Pipe

{{ item.key }} : {{ item.value }}
Converts object into array of key-value pairs for iteration
